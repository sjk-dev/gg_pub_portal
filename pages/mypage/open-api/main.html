Open API 활용 인증키에 대하여

  • 인증키 사용에 대한 자세한 내용은 인증키 이용내역을 통해 확인할 수 있습니다.
  • 정상 상태의 인증키는 최대 15개까지 발급받을 수 있습니다.
  • 최근 2년 동안 이용 실적이 없는 인증키는 자동으로 폐기됩니다.
  • 신청하신 인증키는 타인에게 양도할 수 없습니다.
  • 발급받은 인증키는 관리목적상 활용용도 별로 구분한 것으로 경기데이터드림의 모든 Open API에 활용 가능합니다.
  • 자세한 내용은 ‘개방 > OpenAPI > Open API 사용방법을 참조하시기 바랍니다.
